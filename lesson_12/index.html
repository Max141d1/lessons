transform
Зміщує елемент на нове місце, переміщаючи відносно початкового положення об'єкта вправо та вниз, використовуючи координати X та Y.

scale (x,y)

scale
Масштабує елементи, роблячи їх більшими або меншими. 1 - це початковий розмір об'єкту

rotate (n + deg)

rotate
Повертає елементи.

skew (x + deg, y + deg)

skew
Деформує (нахиляє) елементи.

matrix (a, c, b, d, x, y)

matrix
Дозволяє об'єднати кілька функцій 2D-трансформацій в одній.

Множинні 2D трансформації значення розділяються пробілом

translate rotate scale

Transform-origin

origin
Дозволяє змістити центр трансформації, щодо якого відбувається зміна положення/розміру/форми елемента. transform-origin: 50% 50%; transform-origin: 50%; transform-origin: center;
